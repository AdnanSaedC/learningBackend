comparison operators

    }
})

$lt -less than
$lte -less than equal
$gt 
$gte
db1> db.students.find({age:{$gt:50}})
db1> db.students.find({age:{$gt:50 , $lt:25}})
the range is 25-50 now

the range is 25-50 now
db1> db.students.find({age:{$in[50,25]}})

not in the range
db1> db.students.find({age:{$nin[50,25]}})



logical operators
db.students.find({$and:[{
    fulltime:true
},{
    age:{
        $lte:50
    }
}]})

similarly we have $or
$nor where both conditions has to be falsed opposite of $and

if you want to include null then use $not
db.collectionName.find(
    {
        $age:{
            $not:{
                $gt:30
            }
        }
    }
)

this will give all the documents where the condition is false and also those document where age is null
